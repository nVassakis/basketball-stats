# Source Code Architecture

This is the core pipeline for the Basketmaniacs ML project.

### Execution Order
To run the pipeline from scratch, execute the modules in the following order:
1. `data/scraper.py`
2. `data/parser.py`
3. `data/ingest.py`
4. `features/logic.py`
5. `models/train.py`
